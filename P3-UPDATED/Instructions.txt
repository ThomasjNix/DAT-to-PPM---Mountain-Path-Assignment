Generally, follow the instructions in the terminal. 

1. Enter P3 or P6 (This program is meant to run on P3. This is not intended to run with P6 PPM format!)
2. Enter the file name or the relative path to the file (ex: read_file.dat or ../another_folder/read_file.dat)
3. Enter the file name or the relative path to the file (ex: outputile.ppm or ../another_folder/output.ppm)
4. Enter the number of rows you wish to specify as a startin position
5. For however many number of rows you specified, enter a number between the start and end point (specified in the terminal),
	 seperated by ENTER key presses.
